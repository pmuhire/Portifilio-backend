### Installation

1. Clone this repository
   ```sh
   git clone https://github.com/pmuhire/Portifilio-backend.git
   ```
2. Installing Required Modules
   ```sh
   npm install
   ```
2. Run the project
   ```sh
   npm start
   ```

2. Run tests
   ```sh
   npm test
   ```
